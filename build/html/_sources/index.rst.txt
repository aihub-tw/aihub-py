aihub Python Library
=========================

此說明文件提供 AIHUB Python Library 使用說明。
您可以使用本套件對AIHUB進行演算法與檔案的操作。


Installation
-------------
``aihub`` 套件的安裝可以透過 `pip
<https://pypi.org/project/aihub/>`_
進行
::

    pip install aihub


Example Usage
-------------

::

    from aihub import client

    #define the aihub client
    ai_client = client()
